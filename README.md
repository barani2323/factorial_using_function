# Functions_and_Modulus_in_Python


This repository contains two simple Python programs.

## Task 1: Factorial Program
- Calculates the factorial of a number using a function.
- Example: Factorial of 5 is 120.

## Task 2: Math Module Program
- Takes a number from the user.
- Calculates:
  - Square root
  - Natural logarithm
  - Sine value (in radians)

## Files
- factorial.py
- math_operations.py

## How to Run
Run the programs using Python:
```bash
python factorial.py
